Gameplay FAQ

How do I change control scheme?

From the main menu, tap Controls. Select any of the 4 available schemes — Joystick Left, Joystick Right, D-Pad, or Auto Pilot.

How do waves work?

A new wave begins every 25 seconds. Each wave increases enemy speed, fire rate, and spawn frequency. The game is endless — survive as long as possible.

What are the enemy types?

Fighter (red, fast), Bomber (purple, high HP), and Ace Pilot (orange, homing AI). Ace Pilots appear from Wave 3 onwards.

Does the game work offline?

Yes. Fighter Plane works completely offline. No internet connection required at any point.

Technical Issues

Game is lagging or slow

Close other background apps and restart Fighter Plane. The game targets 60 FPS and is optimized for most devices running iOS 15+ or Android 8+.

Controls are not responding

Go to Menu → Controls and try a different control scheme. If using D-Pad, make sure you are pressing within the button boundaries.

App crashes on launch

Force close the app and relaunch. If the issue persists, reinstall the app. Your game is not tied to an account so no data will be lost.

Keyboard not working (desktop/iPad)

Use WASD or Arrow Keys to move and Space to fire. Press Escape to return to the main menu.
